🎯 5 SIMPLE STEPS
================================================================================

STEP 1: Create GitHub Account (3 minutes)
------------------------------------------
→ Go to: https://github.com/
→ Click "Sign up"
→ Enter email, password, username
→ Verify email
→ Choose "Free" plan

STEP 2: Create Repository (2 minutes)
--------------------------------------
→ Click "+" → "New repository"
→ Name: trading-bot
→ Select: PRIVATE (important!)
→ Check "Add README"
→ Click "Create repository"

STEP 3: Upload Files (5 minutes)
---------------------------------
→ Click "Add file" → "Upload files"
→ Drag and drop ALL your .py files
→ Include: .github/workflows/trading-bot.yml
→ DON'T upload .env file
→ Click "Commit changes"

STEP 4: Add Secrets (5 minutes)
--------------------------------
→ Settings → Secrets and variables → Actions
→ Click "New repository secret"
→ Add these 5 secrets:
   1. SHOONYA_USER_ID
   2. SHOONYA_PASSWORD
   3. SHOONYA_TOTP_KEY
   4. SHOONYA_VENDOR_CODE
   5. SHOONYA_API_SECRET

STEP 5: Enable & Test (5 minutes)
----------------------------------
→ Go to "Actions" tab
→ Enable workflows if needed
→ Click "Run workflow" to test
→ View logs to confirm it works
